# Terminology of Concurrent and Parallel

# Grokking Concurrency

- application layer/应用层
  - coroutines/协程
  - design patterns/设计模式
  - concurrency control/并发控制
  - sequential and order-independent/串行和顺序独立性
- runtime layer/运行时层
  - multitasking/多任务
  - scheduler/调度器
  - process/进程
  - interprocess communication/进程间通信
  - system threads/系统线程
- hardware layer/硬件层
  - instruction-level parallelism/指令集并行
  - cache/缓存
  - serial and parallel execution/串行和并行执行
  - CPU/中央处理器
