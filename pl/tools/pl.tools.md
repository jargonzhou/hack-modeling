# Programming Languages Tools

# Build
* [Ant](https://ant.apache.org/): Apache Ant is a Java library and command-line tool whose mission is to drive processes described in build files as targets and extension points dependent upon each other.
* [Bazel](./tools/Bazel/Bazel.md): a fast, scalable, multi-language and extensible build system
* [Gradle](./Gradle.md): Gradle is the open source build system of choice for Java, Android, and Kotlin developers. From mobile apps to microservices, from small startups to big enterprises, it helps teams deliver better software, faster.
* [Maven](https://maven.apache.org/): Apache Maven is a build tool for Java projects. Using a project object model (POM), Maven manages a project's compilation, testing, and documentation.
